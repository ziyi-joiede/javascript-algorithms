## 24. 两两交换链表中的节点

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

**你不能只是单纯的改变节点内部的值，** 而是需要实际的进行节点交换。

### 示例 1：

![avatar](https://assets.leetcode.com/uploads/2020/10/03/swap_ex1.jpg)

```js
输入：head = [1,2,3,4]
输出：[2,1,4,3]
```

### 示例 2：

```js
输入：head = []
输出：[]
```

### 示例 3：

```js
输入：head = [1]
输出：[1]
```

### 解

```js
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var swapPairs = function(head) {
	if(!head || !head.next) return head;

	let dummy = new ListNode();
	dummy.next = head;
	let curr = head;
	let prev = dummy;
	
	while(curr && curr.next){
		// 交换相邻节点
		let next = curr.next.next;
		prev.next = curr.next;
		curr.next.next = curr;
		curr.next = next;

		// 遍历链表，
		// 跳过已经交换的节点
		curr = next;
		prev = prev.next.next;
	}


	return dummy.next;
};
```
