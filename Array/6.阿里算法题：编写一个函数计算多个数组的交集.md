## 阿里算法题：编写一个函数计算多个数组的交集

### 要求：
输出结果中的每个元素一定是唯一的

```js
const intersection = (...args) => {
	if(args.length===0){
		return [];
	}else if(args.length===1){
		return [...new Set(args[0])];
	}else {
		return [...new Set(args.reduce((acc, arrItem)=>{
			return acc.filter(item => {arrItem.includes(item)}));
		},[]))];
	}
}
```
