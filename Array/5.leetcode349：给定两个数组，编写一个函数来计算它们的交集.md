## 349. 两个数组的交集

#### 示例 1：

```js
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]
```

#### 示例 2：
```js
输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
```

#### 说明：

- 输出结果中的每个元素一定是唯一的。
- 我们可以不考虑输出结果的顺序。

let intersection = (nums1, nums2) => {
	let nums =  nums1.filter(item1=>nums2.includes(item1))
	return [...new Set(nums)];
}
