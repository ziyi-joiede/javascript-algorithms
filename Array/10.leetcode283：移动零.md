## 283. 移动零

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

### 示例:
```js
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```

来源：力扣（LeetCode）

```js
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var moveZeroes = function(nums) {
	let len = nums.length;
	let i = 0,j = 0;
	while(i < len){
		if(nums[i] !== 0){
			[nums[i], nums[j]] = [nums[j], nums[i]];
			i++;
			j++;
		}else{
			i++;
		}
	}
};
```
